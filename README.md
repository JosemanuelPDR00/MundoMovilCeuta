# MundoMovilCeuta

![PaginaInicio](https://github.com/JosemanuelPDR00/MundoMovilCeuta/assets/77966853/ad87159a-bc6c-4eff-b210-cc94081b3f8f)

Pagina web para la tienda MundoMovil de Ceuta, dedicada a la venta de dispositivos moviles y electrónicos. Solicita mis servicios para la creación de una página web multihoja. El objetivo de esta página web es enseñar brevemente los diferentes tipos de productos que ofrecen. Está diseñada para que todos los usuarios que vean un producto que les interese, tengan la posibilidad de contactar a traves de diferentes medios para ver disponibilidad y hacer sus reservas. La web a parte de ser llamativa, es intuitiva para que cualquier usuario pueda realizar un uso correcto y le sea fácil contactar con la tienda.

## Tipografía
Las tipografías sans serif son conocidas por su aspecto limpio y moderno, lo que las hace ideales para su uso en pantallas digitales. Al carecer de remates en las letras, como los que tienen las tipografías serif, son más fáciles de leer en tamaños pequeños y en pantallas de baja resolución. Además, su diseño simple y minimalista las hace muy versátiles y adecuadas para una amplia variedad de aplicaciones, desde diseño web hasta impresión.

## Imagenes
Las imagenes estan Recogidas de las páginas oficiales de las marcas de los dispositivos, con licencia gratuita, de esta manera, podemos obtener fotos mas profesionales y enfocadas en la temática.

## Enlaces
Todos los iconos son enlaces a redes sociales, correo electronico e incuso telefono de contacto, para que cualquier cliente tenga la facilidad de contactarle.

## Diseño
El diseño que hemos elegido ha sido básico, poco recargado y tratando los temas de manera breve pero detalladas. Este diseño es totalmente responsive, con lo cual desde cualquier tipo de dispositivo podriamos tener una agradable utilizacion de la web.

Esta pagina fue creada bajo demanda del cliente, en todo momento hemos estado en contacto con él, para poder aconsejarle y ayudarle. Tener en cuenta que él, ha sido el que ha recalcado siempre cual es la parte mas importante para resaltar de su trabajo, y así dar a conocer sus productos un poco más.
